How Gates Work

Keys are earned by fulfilling conditions; gates unlock at area ends. Blue, Red and Black gates each require playing specific songs from a starting date onward. The tracker shows which songs still need valid play history.

Features

  • Loads all required songs for Blue, Red and Black gates
  • Checks your recent play history to mark valid or missing plays
  • Can add required songs to favorites automatically

Mobile

The easiest way is to create the bookmarklet on PC and sync it. If manual setup is needed:

  1. Copy the bookmarklet code below.
  2. Open your browser menu and add the page to bookmarks.
  3. Edit the bookmark and replace its URL with the copied code.
  4. Rename it to "KALEIDXSCOPE key" or whatever.
